# Laravel vue starter kit with Inertia, TailwindCSS and vue-shadcn-components.
This is a fast starter kit with personal preferences, rdy to start a project.
It is based on laravel vue starter kit with plus features.

## Technologies added
- Inertia
- TailwindCSS
- shadcn-components
- axios
- dayjs (Date handler)

## Benefits
- Composables for manage axios errors.
- Ui custom components 
- Form components (inputs with labels).
